# go-thousandeyes

go-thousandeyes is a [go](https://golang.org/) client library for the [Thousandeyes v6 API](https://developer.thousandeyes.com/v6).

## Installation

First, download the source code
```cli
go get github.com/william20111/go-thousandeyes
```
